# 2. Constraints

## 2.1 Technical Constraints
- The system must run on AWS infrastructure.
- Frontend built using React and TypeScript.
- Database: PostgreSQL 16.
- REST API must comply with OpenAPI 3.0.

## 2.2 Organizational Constraints
- Team of 4 developers, 1 UX designer, 1 DevOps.
- Development must follow Agile Scrum methodology.
- Release every 2 weeks.

## 2.3 Legal / Regulatory Constraints
- Must comply with GDPR (EU data protection).
- Logging data retention: 12 months max.
- All user data encrypted in transit (TLS 1.3).
